# Upgradeable Greenhouse

The **Upgradeable Greenhouse** mod allows you to expand your greenhouse seamlessly through Robin's Carpenter's Shop. Instead of being limited to the default farming area, you can now invest your hard-earned gold and gathered materials to unlock larger, more spacious greenhouse stages. Whether you want to expand your ancient fruit empire or just need more room for trees and kegs, Robin has you covered!

---

## Main Features

* **Immersive Upgrades via Robin:** Purchase greenhouse expansions directly from Robin's shop menu, just like upgrading your house or farm buildings.
* **Dynamic Map Expansion:** The greenhouse map expands dynamically the morning after the construction is finished.
* **Safe Content Shifting:** All your placed items, crops, fruit trees, and furniture are completely safe! The mod uses native game mechanics to automatically preserve and shift your items to fit the new room dimensions.
* **Balanced Progression:** Features multiple purchasable expansion levels with balanced material (Wood, Stone, Hardwood) and gold costs.

---

## Installation Instructions

1. Install the latest version of **[SMAPI](https://smapi.io/)**.
2. Download the Upgradeable Greenhouse `.zip` file from the Files tab (or Releases).
3. Extract the contents of the `.zip` file into your `Stardew Valley/Mods` folder.
4. Launch the game using SMAPI and visit Robin to start upgrading!

---

## Requirements

* **SMAPI - Stardew Modding API:** This is the core framework required to run the mod.
